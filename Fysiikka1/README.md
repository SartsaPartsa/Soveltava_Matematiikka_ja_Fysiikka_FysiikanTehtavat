# Soveltava Matematiikka ja Fysiikka - Fysiikan Tehtävät

Tämä repositorio sisältää fysiikan kurssin tehtäviä ja harjoituksia.

## Sisältö

- `datan_kuvajaa.ipynb` - Jupyter notebook datan visualisointiin ja analyysiin
- `Raw Data.csv` - Raakadata mittauksista
- `Kiihtyvyys_kuva.jpg` - Kiihtyvyyskaavio
- `kuvaajat/` - Kansio graafeille ja kuvaajille
- `meta/` - Metadataa sisältävä kansio
  - `device.csv` - Laitetiedot
  - `time.csv` - Aikaleimoja
- `Tehtava1_palautus_Sara_Vehvilainen.docx/.pdf` - Tehtävä 1 palautus

## Käyttö

1. Avaa `datan_kuvajaa.ipynb` Jupyter Notebookissa datan analysointiin
2. Raakadata löytyy `Raw Data.csv` tiedostosta
3. Kuvaajat ja visualisoinnit tallennetaan `kuvaajat/` kansioon
